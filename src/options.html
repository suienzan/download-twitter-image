availiable variables
  • {name} : Users name.
  • {screen_name} : The one after '@' and in address bar.
  • {id} : Tweet id.
  • {text[$number]} : Fisrt $number characters of tweet content. You will get fisrt 28 characters with `{text[28]}`. USE `{text}` to get all.`
  • {[$pagePrefix]page} : Page number with prefix. With `{[ - ]page}`, you will get a ' - 1' if there are multiple images. And get nothing if there is only one image.
