availiable variables
  • {name} : Users name.
  • {screen_name} : The one after '@' and in address bar.
  • {id} : Tweet id.
  • {[$textPrefix]text[$number]} : Fisrt $number characters of tweet. You will get fisrt 28 characters with `{[ - ]text[28]}`. USE `{[ - ]text}` to get all. Skip this section if no text content.
  • {[$pagePrefix]page} : Page number with prefix. With `{[ - ]page}`, you will get a ' - 1' if there are multiple images. Skip this section if there is only one image.
